# TinyDay 👶

**Every Day is a TinyDay** — One photo, one note, one beautiful memory of your baby's first years.

## Features

- 📅 **Daily Photo Calendar** — See your baby's journey in a beautiful calendar view
- 📸 **One Photo a Day** — Simple, focused, no clutter
- 📜 **Timeline View** — Scroll through memories chronologically
- 💬 **Notes & Milestones** — Capture first words, steps, and everything in between
- 👨‍👩‍👧 **Family Sharing** — Share a private link with grandparents and family
- 📱 **PWA Ready** — Add to home screen for app-like experience

## Tech Stack

- Pure HTML/CSS/JS (zero dependencies)
- LocalStorage for data persistence
- PWA-ready with manifest

## Getting Started

Simply open `index.html` in your browser, or deploy to any static hosting (Vercel, Netlify, etc.).
